# 稜馬のアトリエ — ポートフォリオ vol.2

大崎稜馬（愛知工業大学 情報科学部 情報科学科 メディア情報専攻／水野慎士研究室）のポートフォリオサイトです。
CG・画像処理・Web・映像・企画・グラフィックを横断した全27作品と、活動・人となりをまとめています。

> **vol.1 との違い**
> 前作（[ryoo-ma638.github.io](https://ryoo-ma638.github.io/)）のダーク×スペクトラム基調から一新し、
> 「**つくって、手渡す。**」をテーマに、生成りの紙×紺×琥珀の“明るい仕事場”として全面リニューアルしました。
> 案内役のマスコット「うまくん」（自作キャラクター）がサイト全体を駆け回ります。

## 見どころ

- **うまくん**：ヒーローを2コマアニメで駆け抜け、スクロールに合わせてヘッダーの進捗ラインを走り、フッターではおじぎ。クリックすると紙吹雪が舞います
- **作品詳細（全27ページ）**：ねらい・工夫・ギャラリー・スライド・音源つきのケーススタディ。一覧から画像がつながるView Transitions
- **売り子カレンダー**：バンテリンドームのビール売り子としての出勤予定を球団スケジュール型で
- **YUMIKI WORLD**：著作権・肖像権に配慮して制作した非公式ファンサイト（Web作品として収録・noindex）

## 技術

- [Astro](https://astro.build/)（静的サイト生成）＋ TypeScript
- View Transitions／IntersectionObserver による省電力な動画・アニメーション制御
- デザインシステムは `DESIGN.md`、開発用QAツールは `qa-crawl.mjs`（全ページ自動検査）・`shot.mjs`（スクリーンショット）

## 開発

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # dist/ に静的出力
node qa-crawl.mjs  # 全ページQA（要: ビルド済み + preview起動）
```
